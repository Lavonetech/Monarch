# Monarch

## Back-end

### Installation

```bash
npm install
```

### Execute
```bash
npm run dev
```

## .env Format
```
PORT=
MONGODB_URI=
```
## ENDPOINT
Add Parts
- name: text
- image: file
```
/v1/clothe/main-part
```

Add Sub Parts
- name: text
- image: file
- id: text
```
/v1/clothe/sub-part
```

## Front-end

### Installation

```bash
npm install
```

### Execute
```bash
npm start
```
